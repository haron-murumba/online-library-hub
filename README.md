# Online Maktaba - Digital Library for Students

A comprehensive digital library platform for primary and secondary school students to access learning resources.

## Features

- User authentication and profiles
- Book management system
- Study resources (worksheets, notes, videos, quizzes)
- Search functionality
- Responsive Bootstrap design
- File upload and download system
